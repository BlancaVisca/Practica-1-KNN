# Práctica 1 – Clasificador K-NN (Iris Dataset)

Este repositorio contiene el desarrollo de la **Práctica 1**, cuyo objetivo es implementar un **clasificador supervisado K-Nearest Neighbors (K-NN)** aplicado al conjunto de datos **Iris**.  

El proyecto incluye una implementación en **Python** con interfaz gráfica (GUI) para cargar archivos, seleccionar el valor de **K**, evaluar la exactitud del modelo y clasificar datos desconocidos, además de una validación de resultados mediante la herramienta de minería de datos **WEKA**.

## Funcionalidades principales
- Carga de archivos de **entrenamiento**, **prueba** y **datos nuevos**.
- Clasificación mediante **K-NN** usando distancia euclidiana.
- Cálculo de **exactitud** para diferentes valores de \(K\).
- Exportación de resultados en formato **Excel (.xlsx)**.
- Comparación de desempeño con la herramienta **WEKA (IBk)**.
